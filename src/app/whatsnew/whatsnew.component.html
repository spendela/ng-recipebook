Version 6.0

Features

  • Will be adding http access to firebase.
  • http.post -> posts to the data to the db, while appending the previous data.
  • http.put -> puts to the data to the db, its just creates new data
  • http.get -> gets the data to the db
  • each http.get or http.put returns an observablee, so make sure you subscribe.
  • one call can have multiple or sequential puts or get. So, to emit multiple
    observables in one return call statement, use combineLatest() ( e.g: combineLatest(responseObservable) ) from rxjs.
    Note: you should handle multiple return responses in your subscribe() method.
